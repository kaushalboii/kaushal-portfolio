# 🌌 M Kaushal Maran — Premium Portfolio

A high-performance, visually stunning interactive 3D portfolio built for **M Kaushal Maran**, a Mobile App Developer and B.Tech Information Technology student at SIMATS Engineering College.

🔗 **Live Preview**: [kaushalmaran-portfolio.netlify.app](https://kaushalmaran-portfolio.netlify.app/)

## 🚀 Key Features

- **Interactive 3D Background**: A custom-built Three.js particle field featuring a subtle signature "M" pattern and thematic "Tech Artifacts" (Torus, Dodecahedron, etc.) representing different project domains (Wellness, Law, AI, Mobile).
- **Premium Aesthetics**: Dark-mode-first design with high-contrast typography and subtle glassmorphism effects.
- **Dynamic Projects Section**: Showcasing live mobile applications on both the App Store and Google Play Store.
- **Custom Cursor & Reveal Effects**: Smooth UI interactions and scroll-triggered animations for an immersive experience.
- **Fully Responsive**: Optimized for seamless viewing across smartphones, tablets, and desktops.

## 🛠️ Tech Stack

- **Core**: HTML5, Vanilla JavaScript, CSS3
- **3D Graphics**: Three.js (WebGL)
- **Typography**: Google Fonts (Playfair Display, Outfit, Space Mono)
- **Icons**: Emoji & Custom CSS Shapes
